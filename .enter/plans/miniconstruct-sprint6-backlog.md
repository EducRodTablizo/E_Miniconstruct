## 1. Database Migration (single script)

### A. `historical_reports` table
```sql
CREATE TABLE historical_reports (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  report_type TEXT NOT NULL CHECK (report_type IN ('daily','weekly','monthly','yearly')),
  period_label TEXT NOT NULL,          -- e.g. "2026-06-03", "2026-W23", "2026-06", "2026"
  period_start TIMESTAMPTZ NOT NULL,
  period_end   TIMESTAMPTZ NOT NULL,
  total_sales  NUMERIC NOT NULL DEFAULT 0,
  total_transactions INTEGER NOT NULL DEFAULT 0,
  inventory_changes JSONB DEFAULT '[]',
  user_activities   JSONB DEFAULT '[]',
  audit_summary     JSONB DEFAULT '[]',
  generated_by UUID REFERENCES profiles(id),
  created_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE historical_reports ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Privileged users can manage historical_reports" ON historical_reports
  FOR ALL TO authenticated USING (is_owner()) WITH CHECK (is_owner());
```

## 5. AI Edge Function

### `supabase/functions/inventory-assistant/index.ts`
1. Verify caller is authenticated
2. Parse user message from request body
3. Fetch context data from DB:
   - All products with stock levels and categories
   - Last 30 days of transactions with totals
   - Low-stock items (stock ≤ reorder_level)
4. Call Enter AI API (Anthropic Claude via `AI_API_TOKEN`) with:
   - System prompt: "You are an inventory assistant for MiniConstruct construction materials. Answer questions about inventory, sales, and stock levels. Format tables in markdown."
   - Context: structured inventory + sales data
   - User's question
5. Return streamed or complete text response
6. Log query to audit_logs via `log_audit_event` RPC

## 6. New Pages

### `src/pages/AIAssistantPage.tsx`
- Chat interface with message history (state-managed, not persisted)
- Text input + Send button
- Typing indicator during AI call
- Renders markdown responses (using a simple markdown renderer)
- Suggested queries as chips: "Show low stock items", "Today's sales summary", "Which products need reordering?"
- Calls `supabase.functions.invoke('inventory-assistant', { body: { message } })`
- RBAC: accessible to all active users (staff + owner + admin)

### `src/pages/HistoricalReportsPage.tsx` (owner/admin only)
- Top: "Generate Report" form with:
  - Report type selector (daily/weekly/monthly/yearly)
  - Date range picker (start date + end date inputs)
  - "Generate & Save" button → calls `generate-historical-report` edge function
- Below: table of saved reports with columns: Type, Period, Total Sales, Total Transactions, Generated By, Date
- "View" button per row → opens a dialog showing:
  - Summary stats
  - Inventory Changes list
  - User Activities list
  - Audit Log summary

### `supabase/functions/generate-historical-report/index.ts`
1. Verify caller is owner/admin
2. Parse report_type, period_start, period_end
3. Query transactions in range for total_sales, total_transactions
4. Query audit_logs in range for user_activities and audit_summary
5. Query products for inventory snapshot
6. INSERT into historical_reports
7. Return the saved report
